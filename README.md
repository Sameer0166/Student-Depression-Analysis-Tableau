# Student Depression Analysis Dashboard

## Project Overview

The **Student Depression Analysis Dashboard** is an interactive Tableau project developed to explore factors associated with student mental well-being. Using survey-based data, the dashboard analyzes academic pressure, financial stress, sleep duration, study habits, and satisfaction levels to identify patterns that may contribute to depression among students.

The project demonstrates how Tableau can transform survey data into interactive visualizations that support data-driven understanding of mental health trends.

---

## Business Problem

Educational institutions and researchers require meaningful insights into student well-being to better understand the challenges affecting academic performance and mental health.

This project helps answer questions such as:

- How does academic pressure affect students?
- Does financial stress influence depression levels?
- How are study habits related to student well-being?
- What role does sleep duration play?
- Which factors appear most closely associated with depression?
